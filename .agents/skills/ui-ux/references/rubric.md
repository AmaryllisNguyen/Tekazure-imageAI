# UI UX Rubric (0-5)

Score each category from 0 to 5.

## 1) Visual Hierarchy
- 0: no clear structure, cluttered.
- 3: acceptable structure, primary action visible.
- 5: instantly scannable, strong emphasis and rhythm.

## 2) Usability
- 0: core tasks are confusing or blocked.
- 3: main task is doable with minor friction.
- 5: core flow feels obvious and fast.

## 3) Visual Craft
- 0: generic/template-like and inconsistent.
- 3: coherent style with limited distinctiveness.
- 5: memorable, brand-fit, polished, intentional.

## 4) Responsiveness
- 0: breaks on common screen sizes.
- 3: works across screens with minor layout issues.
- 5: optimized experience for mobile, tablet, desktop.

## 5) Accessibility Basics
- 0: poor contrast/focus/labels.
- 3: mostly accessible with a few gaps.
- 5: solid keyboard, contrast, semantics, states.

## 6) Product Outcome Alignment
- 0: visuals do not support user/business goal.
- 3: partial alignment with measurable outcomes.
- 5: design directly supports conversion and retention metrics.

## Quality gate
- Minimum acceptable: 4/5 on Hierarchy, Usability, Responsiveness, Visual Craft.
- Stretch target: average 4.5+ across all categories.
